CS Exam Study

Q.1 In the context of computer security, crackers are also known as:

(a) black hat hackers
(b) white hat hackers
(c) elite hackers
(d) script kiddie

Right Ans: (a) black hat hackers

Q.2 DuckDuckGo is a:

(a) search engine
(b) web browser
(c) virus
(d) news web site

Right Ans: (a) search engine

Q.3 Ethernet is an example of:

(a) MAN
(b) LAN
(c) WAN
(d) WiFi

Right Ans: (b) LAN


Q.4 Weibo is a social media platform popularly used in:

(a) South Korea
(b) Japan
(c) Thiland
(d) China

Right Ans: (d) China

Q.5 Which of the following is an extremely fast, small memory between CPU and main memory?

(a) Main RAM and ROM
(b) Cache Memory
(c) Secondary Memory
(d) None of the above

Right Ans: (b) Cache Memory
