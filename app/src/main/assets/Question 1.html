CS Exam Study

Q.1 Which is the part of computer system that one can physically touch?

(a) Data
(b) Operating system
(c) Hardware
(d) Software

Right Ans: (c) hardware

Q.2 IC chips used in computers are usually made of:

(a) Lead
(b) Silicon
(c) Chromium
(d) Gold

Right Ans: (b) Silicon

Q.3 Which of the following is not an example of an Operating System?

(a) Windows 98
(b) BSD Unix
(c) Microsoft Office XP
(d) Red Hat Linux

Right Ans: (c) Microsoft Office XP

Q.4 One Gigabyte is approximately equal is:

(a) 1000,000 bytes
(b) 1000,000,000 bytes
(c) 1000,000,000,000 bytes
(d) None of these

Right Ans: (b) 1000,000,000 bytes

Q.5 Compact discs, (according to the original CD specifications) hold how many minutes of music?

(a) 74 mins
(b) 90 mins
(c) 56 mins
(d) 60 mins

Right Ans: (a) 74 mins
